# MableLink

MableLink is a professional AI pipeline platform designed to connect Digital Content Creation (DCC) applications with AI workflows.

Current Targets

- Autodesk Maya
- Blender
- ComfyUI

Mission

Allow artists to stay inside their favorite DCC application while seamlessly leveraging modern AI tools.
